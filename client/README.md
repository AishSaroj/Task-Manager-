# 📝 Task Tracker MERN App

A full-stack Task Management web application built using the MERN Stack (MongoDB, Express, React, Node.js).  
It helps users create, manage, update, and track tasks with a clean and responsive UI.

---

## 🚀 Features

- ➕ Create new tasks (title, description, priority, due date)
- 📋 View all tasks in dashboard format
- ✏️ Edit tasks
- 🗑️ Delete tasks
- ✅ Mark tasks as completed / pending
- 🔔 Pending task notification badge
- 📊 Task statistics dashboard
- 🔍 Search functionality (UI ready)
- 📱 Fully responsive design
- ⚡ Instant updates without page refresh

---

## 🛠️ Tech Stack

**Frontend:** React.js, Tailwind CSS, Axios, React Icons  
**Backend:** Node.js, Express.js  
**Database:** MongoDB, Mongoose
